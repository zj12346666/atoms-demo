## 🚀 快速开始

### 本地运行

```bash
# 1. 安装依赖
npm install

# 2. 配置环境变量
cp .env.example .env.local
# 编辑 .env.local，填入 OPENAI_API_KEY

# 3. 启动开发服务器
npm run dev

# 4. 访问应用
open http://localhost:3000
```

### 部署到 Vercel

```bash
# 方式一：通过 Vercel Dashboard
# 1. 推送代码到 GitHub
# 2. 访问 vercel.com/new
# 3. 导入项目
# 4. 配置环境变量 OPENAI_API_KEY
# 5. 点击部署

# 方式二：通过 CLI
vercel --prod
```

详细部署指南见 `DEPLOYMENT.md`
