## ⚠️ 重要：配置 API Key

在测试完整功能前，需要配置 OpenAI API Key：

### 步骤 1：创建环境变量文件

```bash
cd /Users/zhangjie/atoms-demo/atoms-demo
touch .env.local
```

### 步骤 2：编辑 `.env.local`

```env
OPENAI_API_KEY=sk-your-actual-api-key-here
```

> 💡 如何获取 API Key：访问 [platform.openai.com](https://platform.openai.com/api-keys)

### 步骤 3：重启开发服务器

```bash
# 停止当前服务器（Ctrl+C）
# 然后重新启动：
npm run dev
```

## 🐛 常见问题

### Q: 点击发送后一直显示"生成中..."

**原因**：API Key 未配置或无效

**解决**：
1. 检查 `.env.local` 文件是否存在
2. 确认 API Key 格式正确（以 `sk-` 开头）
3. 重启开发服务器
4. 打开浏览器控制台查看错误信息

### Q: 预览区域显示空白

**原因**：生成的代码可能有错误

**解决**：
1. 查看预览区域底部的 Console 输出
2. 检查是否有 JavaScript 错误
3. 尝试更详细地描述需求

### Q: 显示 "Missing credentials" 错误

**原因**：环境变量未加载

**解决**：
```bash
# 确保在项目根目录
cd /Users/zhangjie/atoms-demo/atoms-demo

# 检查文件是否存在
ls -la .env.local

# 确认文件内容
cat .env.local

# 重启服务器
npm run dev
```
